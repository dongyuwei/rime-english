# rime-english
custom config for [rime-english](https://github.com/sdadonkey/rime-english)

配置文件主入口是 `default.custom.yaml` 和 `rime.lua`，mac路径为 `~/Library/Rime/`。

luna_pinyin.custom.yaml 只是配合 date_translator 演示一下 librime-lua 插件的能力。
